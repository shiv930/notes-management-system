# 📝 Notes Management System

A full-stack Notes Management Application built using the MERN Stack. Users can securely register, login, create, update, delete, and manage personal notes with authentication and protected routes.

## 🚀 Features

- User Authentication (JWT)
- Secure Login & Register
- Protected Routes
- User Profile
- Create Notes
- Read Notes
- Update Notes
- Delete Notes
- Redux Toolkit State Management
- Cookie-based Authentication
- Multer Image Upload
- ImageKit Cloud Storage
- Responsive UI
- REST API

## 🛠 Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt
- Multer
- ImageKit
- Cookie Parser
